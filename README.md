zsh-peepcodelike
================

A Peepcode-like ZSH theme based on [John Barker](https://github.com/excepttheweasel)'s theme, adjusted for a dark theme (like OSX Pro).

Installation
----------------
* Copy the theme in ```~/.oh-my-zsh/themes```
* Edit ~/.zshrc
